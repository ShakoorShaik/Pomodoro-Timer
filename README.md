## Description

A simple pomodoro timer that can be used to track your work hours. It was created using HTML, CSS, and JavaScript. 

## Features

- Tracks work hours
- Start, stop, and reset the timer

```
Options:
1. Start
2. Stop
3. Reset
```

## Technologies Used
- HTML5
- CSS3
- JavaScript
